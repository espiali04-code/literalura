# 📚 Literalura

Aplicación desarrollada en **Java con Spring Boot** que permite consultar libros desde la API pública **Gutendex** y almacenarlos en una base de datos **PostgreSQL**.

## 🚀 Funcionalidades

- Buscar libros por título usando la API Gutendex
- Guardar libros en la base de datos
- Listar libros guardados
- Listar autores registrados
- Consultar autores vivos en un año específico
- Filtrar libros por idioma

## 🛠 Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- API Gutendex

## 📡 API utilizada

https://gutendex.com/

## ▶ Cómo ejecutar el proyecto

1. Clonar el repositorio:
git clone https://github.com/espiali04-code/literalura.git
2. Configurar PostgreSQL en `application.properties`.

3. Ejecutar la aplicación desde IntelliJ o con:
mvn spring-boot:run
